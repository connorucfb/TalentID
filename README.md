# Talent Identification Project 

This project explores player recruitment using football performance data.

The research aim was to identify players who have the potential to play at a higher level.

## Objective

Develop a Python workflow capable of analysing player statistics to support recruitment decisions through objective data analysis and visualisations.

## Repository Structure

- **Talent Identification and Recruitment.pptx** – Final project presentation summarising the methodology, analysis and key findings.
- **TalentID.ipynb** – Main analysis notebook used to identify high-performing players from League One using statistical performance data.
- **data/** – Folder containing the League One, Championship and Premier League datasets used throughout the project.
- **poku_radar.ipynb** – Radar visualisation comparing Kwame Poku against Championship wingers.
- **bielik_radar.ipynb** – Radar visualisation comparing Krystian Bielik against Championship defenders.
- **coventry_radar.ipynb** – Radar visualisation comparing Conor Coventry against Championship midfielders.
